# Heart Disease Prediction System

## About This Project
A machine learning system that predicts heart disease risk with 88.2% accuracy.

## Features
- Interactive data visualizations
- 5 ML algorithms comparison
- Real-time prediction web app
- Feature importance analysis

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- Pandas
- Matplotlib

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run streamlit_app.py`
